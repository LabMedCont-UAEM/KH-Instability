Python codes for Kelvin-Helmholtz Instability, the unperturbed main flow moves by an electromagnetic force using a Quasi 2D model.
The file "Chebyv2.py" must to be in the same directory that "Lorentz_force_Q2D.py" or "Qcrit.py"

We run it on conda, library needed:
-SciPy (for linalg and "erf" function)
-Numpy (for trigonometric functions)
-Matplotlib (for graphs)
-csv (for saving)

Codes from our paper entitled "Electromagnetically generated Kelvin-Helmholtz instability in a narrow channel"
